# Кодер и декодер Шеннона

Проект для кодирования и декодирования информации с использованием алгоритма Шеннона.

## Описание

Этот проект представляет собой реализацию алгоритма Шеннона для сжатия данных на C++. Алгоритм использует статистическое кодирование для создания оптимальных кодов для каждого символа в исходном тексте/картинке.

## Структура проекта

- `src/` - исходный код проекта
  - `main.cpp` - точка входа программы
  - `filemanager.cpp` - управление файлами
  - `wordlist.cpp` - работа со списком слов
  - `encoder.cpp` - реализация кодировщика
  - `decoder.cpp` - реализация декодера
- `include/` - заголовочные файлы
- `tests/` - модульные тесты
- `Makefile` - сборка проекта

## Требования

- C++17 совместимый компилятор (g++ или clang++)
- Make
- Google Test (для запуска тестов)

## Сборка

```bash
# Сборка основного проекта
make

# Запуск тестов
make test
```

## Использование

После сборки проекта:

```bash
# Кодирование файла
./shannon -encode <входной_файл> <выходной_файл> <"название словаря".txt>
./shannon -e <входной_файл> <выходной_файл> <"название словаря".txt>
# Декодирование файла
./shannon -decode <закодированный_файл> <"название словаря".txt> <выходной_файл>
./shannon -d <закодированный_файл> <"название словаря".txt> <выходной_файл>
```

## Очистка

```bash
# Удаление объектных файлов
make clean

# Удаление всех скомпилированных файлов
make cleanall
```
